# Skills Directory

Skills are specialized capabilities you can load on demand with `/skill <name>`.

## Available Skills

- `code-review` — Review code for bugs, style, and improvements
- `security-audit` — Security analysis and recommendations  
- `bug-triage` — Analyze and prioritize bug reports
- `release-checklist` — Pre-release verification steps
- `perf-profiling` — Performance analysis guidance
- `docs-writer` — Documentation writing assistance
- `migration-planner` — Plan and execute migrations
- `incident-response` — Handle production incidents

## Using Skills

In chat, type: `/skill code-review`

The skill content will be loaded into context for that conversation.
